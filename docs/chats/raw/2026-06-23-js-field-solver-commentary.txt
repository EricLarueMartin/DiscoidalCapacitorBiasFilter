# Raw Conversation Record: JavaScript Field Solver Commentary

Date: 2026-06-23

User: Asked what the iterations count means and whether `solveField` in `app.js` is actually doing FEA in JavaScript. User noticed something like a half-stepping or relaxation algorithm but no meshing, and asked for copious code comments.

Assistant outcome: Confirmed the browser fallback is not a full FEA solver. It builds a fixed, uniformly spaced axisymmetric r-z grid, samples conductors and dielectric regions onto that grid, then uses Gauss-Seidel-style relaxation on a finite-difference / finite-volume-like stencil for `div(epsilon grad V) = 0`. The displayed count is the number of full relaxation sweeps on that fixed grid. Added detailed comments to the solver code, clarified the UI label as `Solver sweeps`, and fixed the reported sweep count so it no longer over-reports by one when the maximum sweep limit is reached.

Follow-up local-refinement note: User said the geometry is simple enough that browser Laplacian relaxation should converge well, but the existing grid spacing is too large around corners. User requested spacing near corners no larger than one tenth of the rounded-edge radius. Updated the browser fallback from a uniform grid to a structured nonuniform grid: it keeps base coordinates across the whole domain, injects local r and z coordinates around the rounded inner ground-plate edges and rounded outer HV-plate edges, and uses local spacing in the relaxation stencil and field extraction. The heatmap renderer now draws nonuniform cells at their actual extents. Browser verification after reload showed the Solve button completing through the JS fallback with a finite peak field and no console errors.

Follow-up adaptive-refinement note: User said low-field regions do not need high accuracy and suggested subdividing where the field is high. Added a field-driven adaptive pass to the browser fallback. The solver now runs a short probe relaxation on the base edge-refined grid, finds high-field dielectric nodes, inserts midpoint coordinates around selected high-field regions, seeds the refined grid by interpolating the probe voltage, and then runs the final relaxation. Browser verification with all four spacing/thickness controls set to 1 mm completed with the Model readout showing `JS adaptive`, reporting a finite peak field and no console errors.
