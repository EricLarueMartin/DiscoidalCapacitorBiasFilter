# Agent Guide: docs/chats/raw

This directory is for raw AI collaboration records.

Preserve original wording where practical. Use filenames beginning with `YYYY-MM-DD` and include a short topic slug.

If a conversation was not exported verbatim at the time, create a clearly labeled reconstructed raw record from the visible chat context. Note that intermediate tool steps were omitted and that some user turns may respond to partially completed work, which can make the record read like a set of non sequiturs.
