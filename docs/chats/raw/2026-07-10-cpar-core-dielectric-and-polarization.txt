Assistant:
Confirmed that the ordinary field-stress solve should retain the equipotential HV core boundary, while the separate adjacent-stage capacitance solve must mesh the core as dielectric. The existing Cpar path already attempted that distinction, but two additional problems were found.

First, the old energy-difference method inferred middle-plate capacitance to ground from the three-plate average. In the default case it subtracted 33.7663 pF from 34.0636 pF and divided by two, producing 0.1486 pF. A roughly one-percent edge-versus-middle ground-capacitance difference therefore removed a large fraction of the desired mutual signal. The solver was changed to three energy solves (middle only, neighbors only, and all bias plates), using polarization to cancel ground terms algebraically.

Second, the capacitance mesh left dielectric inside each bias plate's axial thickness and drove the core only from the plate's cylindrical inner edge. The capacitance-only bias electrodes were extended across the core cross-section while leaving the core segments between plates dielectric. The default result increased to 0.1983 pF at the normal mesh setting.

Mesh refinement gave 0.1983, 0.2005, and 0.2009 pF for edge-radius mesh ratios 0.20, 0.10, and 0.05. A core-permittivity sweep also confirmed that the core was active dielectric. The remaining difference from the larger analytic core-plus-epoxy parallel-plate value is attributed mainly to shielding by the intervening ground-plate inner edge; the simple epoxy area-over-distance estimate does not model that interception of field lines.
